# 개념정리

* select
 + table에서 호출할 columns을 조회 때 사용
```sql
SELECT 컬럼1, 컬럼2
FROM 테이블명
```
* orderby
  + ORDER BY 정렬할 컬럼 ASC  → 오름차순
  + ORDER BY 정렬할 컬럼 DESC → 내림차순
```sql
SELECT 컬럼1, 컬럼2
FROM 테이블명
ORDER BY 컬럼1 ASC 
-- ORDER BY 컬럼1 DESC
```
